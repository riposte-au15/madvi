MadVI — Madrigal Virtual Intelligence

MadVI is a multi-tool scientific copilot built with Langflow that can:

Generate runnable globalIsprint.py Madrigal commands

Search official Madrigal documentation

Recommend scientific papers from a structured Q1–Q7 paper table

Search and explain ingested Madrigal-related PDFs

It uses:

Langflow

ChromaDB (vector database)

Custom tools (Python components)

Madrigal Web API
